# NovaNews - Agregador Inteligente de Noticias Personalizadas mediante IA Generativa

Este Trabajo de Fin de Grado (TFG) del Grado en Ingeniería Informática de la Universidad de La Laguna, correspondiente al curso 2025-2026, ha sido desarrollado por Daniel Carbonell González de Chaves bajo la tutorización de Elena Sánchez Nielsen.

---

## Descripción del Proyecto

Este repositorio contiene el código fuente de NovaNews, una plataforma web diseñada para combatir la sobrecarga de información en internet. El sistema funciona como un agregador de noticias que recopila artículos de diferentes fuentes de forma automática, los clasifica mediante técnicas de búsqueda semántica y utiliza Inteligencia Artificial generativa (modelos de lenguaje ejecutados en local) para redactar resúmenes personalizados que se adaptan a los intereses y al historial de lectura de cada usuario.
